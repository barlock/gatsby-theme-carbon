<PageDescription>


When authoring markdown using the Carbon Gatsby theme, code blocks have some
extra super powers you can take advantage of. We provide carbon-themed syntax
highlighting as well as optional `path`, `src`, and `overflow` features.

</PageDescription>

<Title>Configuring Vertical overflow</Title>

You can specify after how many lines you want your code to overflow. Set
`overflow` to a number like you do for `path` or `src`.

```markdown overflow=5
## Overflow set to 5

This example demonstrates the show more button. This example demonstrates the
show more button. This example demonstrates the show more button. This example
demonstrates the show more button. This example demonstrates the show more
button. This example demonstrates the show more button. This example
demonstrates the show more button. This example demonstrates the show more
button. This example demonstrates the show more button. This example
demonstrates the show more button. This example demonstrates the show more
button. This example demonstrates the show more button.
```

````

<Title>Disabling Vertical overflow</Title>

Additionally, you can disable overflow completely by setting `overflow` to `0`.

```markdown overflow=0
## Overflow disabled

This example demonstrates no show more button. This example demonstrates no show
more button. This example demonstrates no show more button. This example
demonstrates no show more button. This example demonstrates no show more button.
This example demonstrates no show more button.
```

````
```markdown overflow=0
````

### Props

| property | propType | description                                                                                                                 |
| -------- | -------- | --------------------------------------------------------------------------------------------------------------------------- |
| language | string   | [Available languages.](https://github.com/FormidableLabs/prism-react-renderer/blob/master/src/vendor/prism/includeLangs.js) |
| src      | string   | The full url of a relevant link (source)                                                                                    |
| path     | string   | A string indicating the filename or path. Due to markdown limitations this can only be a single word                        |
| overflow | number   | A number indicating how many lines to display before overflowing. Use `0` to disable overflowing completely.                |
